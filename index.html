How to add your name to this page

This page is community-maintained. Add your profile by opening a Pull Request. The site automatically discovers and displays all JSON files in the students/ folder (excluding the template). 

# Fork the original repository
git clone https://github.com/<your_username>/jubcse2026.github.io.git
cd jubcse2026.github.io
cp students/template.json students/your-roll.json
# edit students/your-roll.json with your details
# add your photo to images/
git add .
git commit -m "Add profile: your name"
git push
# Make a PR on GitHub

Full instructions are available in the repository README.
